# 🚀 Spring Boot 3: Desenvolva uma API REST em Java

Este repositório contém anotações, aprendizados e possíveis implementações práticas baseadas no curso:

🔗 https://cursos.alura.com.br/course/spring-boot-3-desenvolva-api-rest-java

---

## 📚 Sobre o Curso

O curso tem como objetivo ensinar, de forma prática, como construir uma API REST utilizando **Spring Boot 3** e **Java**, seguindo boas práticas de desenvolvimento e organização de código.

---

## 🧠 Conteúdos Abordados

### 🔹 Criação de API REST
- Estrutura básica de um projeto com Spring Boot
- Criação de endpoints REST (`@RestController`)
- Boas práticas de organização de código

### 🔹 Persistência de Dados
- Integração com banco de dados
- Uso do **Spring Data JPA**
- Criação de entidades e repositórios

### 🔹 CRUD Completo
- Criação (POST)
- Leitura (GET)
- Atualização (PUT)
- Remoção (DELETE)

### 🔹 Validações
- Validação de dados com **Bean Validation**
- Uso de anotações como `@Valid`, `@NotNull`, etc.

### 🔹 Boas Práticas
- Separação em camadas (Controller, Service, Repository)
- Uso de DTOs (Data Transfer Objects)
- Tratamento de erros

---

## 🛠️ Tecnologias Utilizadas

- Java 17+
- Spring Boot 3
- Maven
- Spring Data JPA
- Hibernate
- Banco de dados (ex: H2, PostgreSQL)
